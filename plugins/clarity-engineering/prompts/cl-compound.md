---
description: Curate accumulated Clarity Engineering learnings
argument-hint: "[completed work, decision, or learning]"
---

Apply Clarity Engineering Compound mode. Continuous compounding already captures learnings at the end of every Build. This standalone Compound command curates accumulated learnings: review, consolidate duplicates, refresh stale docs, promote reusable patterns, and archive what's no longer relevant.

For each accumulated learning or flagged doc, decide: promote, refresh, deduplicate, archive, keep, or no action. Use the memory destination rule: codebase-specific facts → repo memory, cross-repo patterns → global memory, framework changes → framework vault.

Search before writing to avoid duplicates. Escalate with one focused question if the reuse value or target location is unclear. Include curation decision, memory destination, and Compound progress status.

Input:

$ARGUMENTS
